# Important types of mathematical statements

- **Universal statement**: A property holds for every element in a set.
  - Example: “All positive numbers are greater than zero.”
  - See: [Universal statement](Universal statement.md)

- **Existential statement**: There is at least one object for which the property is true.
  - Example: “There is a prime number that is even.”
  - See: [Existential statement](Existential statement.md)

- **Conditional statement**: If one statement is true, then another statement must also be true.
  - Example: “If 378 is divisible by 18, then 378 is divisible by 6.”
  - See: [conditional statements](conditional statements.md)

- **Quantifier order statements**: Some statements mix `∃` and `∀`.
  - See: [Existential-Universal vs Universal-Existential Statements](Existential-Universal vs Universal-Existential Statements.md)
  - See: [Existential Universal Statements](Existential Universal Statements.md)
  - See: [Universal Existential Statements](Universal Existential Statements.md)
