# Second Brain in VS Code

This folder now serves as a lightweight, Git-backed knowledge base for work notes, reading notes, and study notes.

## Structure
- Inbox/ - raw ideas and quick capture
- Work/ - work notes and project capture
- Kindle/ - book highlights and reading summaries
- Discrete Math/ - study notes and problem work
- Templates/ - reusable note templates
- Assets/ - images and other attachments

## Suggested workflow
1. Capture ideas in Inbox.
2. Move useful notes into the relevant section.
3. Keep notes short and structured.
4. Commit changes regularly so you have a backup and history.
