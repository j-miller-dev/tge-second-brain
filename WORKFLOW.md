# VS Code Second Brain Workflow

This workspace is designed as a Git-backed note system for work, reading, and study.

## Recommended structure
- Inbox/ - quick capture
- Work/ - work notes, projects, decisions
- Kindle/ - book highlights and summaries
- Discrete Math/ - study notes and problems
- Templates/ - reusable note scaffolds

## Daily routine
1. Open the workspace in VS Code.
2. Add a quick note in Inbox/.
3. Move useful ideas into the relevant section.
4. Commit changes regularly with Git.

## Git usage
```bash
git status
git add .
git commit -m "Update notes"
git push
```
