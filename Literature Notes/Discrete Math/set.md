# Set

A set is a collection of distinct objects, called elements, treated as a single entity.

Example:

- `A = {1, 2, 3}` is the set containing the elements 1, 2, and 3.

Key idea:

- Each element either belongs to the set or does not.
- Order and repetition do not matter: `{1, 2, 3}` is the same as `{3, 2, 1}`.
