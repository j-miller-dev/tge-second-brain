Use variables to rewrite the following sentences more formally.

## a. Existential statement
Original sentence:

> Are there numbers with the property that the sum of their squares equals the square of their sum?

A clear formal version is:

> Do there exist numbers `a` and `b` such that `a^2 + b^2 = (a + b)^2`?

This uses the variable names `a` and `b` and the symbol `∃` for “there exists”: 

> `∃a ∃b (a^2 + b^2 = (a + b)^2)`

The answer is yes. One example is:

- `a = 5`, `b = 0`

Then:

- `a^2 + b^2 = 5^2 + 0^2 = 25`
- `(a + b)^2 = (5 + 0)^2 = 25`

So the equation holds.

## b. Universal statement
Original sentence:

> Given any real number, its square is nonnegative.

A clear formal version is:

> For every real number `r`, `r^2 ≥ 0`.

This uses the variable `r` and the symbol `∀` for “for every”:

> `∀r ∈ ℝ (r^2 ≥ 0)`

Alternative phrasing:

- “For any real number `r`, `r^2 ≥ 0`.”
- “For every real number `r`, `r^2` is nonnegative.”

## Notes on the notation
- `∃` means “there exists” and introduces an example or witness.
- `∀` means “for all” and shows the statement applies to every value in a set.
- Variables like `a`, `b`, and `r` are placeholders for numbers.

Using these variables makes the meanings of the original sentences precise and easy to work with.
