# Universal-Existential Statements

A universal-existential statement has the form:

- `∀y ∃x P(x, y)`

Meaning:

- For every `y`, there exists an `x` that satisfies the property.

Example:

- “For every problem, there is a student who solved it.”
- Symbolically: `∀p ∃s (Solved(s, p))`

This is different from `∃x ∀y P(x, y)`, which requires one `x` that works for all `y`.
